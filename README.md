# Kokoa Talk

React Native Chat App built with Typescript and Prisma

## User Stories

- [ ] Can create account
- [ ] Can see my profile
- [ ] Can edit profile
- [ ] Can search for people
- [ ] Can see people's profile
- [ ] Can add people as friends

## Message Stories

- [ ] Can create a chat
- [ ] Can send a message
- [ ] Can see chats
- [ ] Can see chat
